# basics

这个目录用于存放 [A Tour of Go - List](https://go.dev/tour/list) 中 `basics` 部分的跟练代码。

## 目录用途

- 按照 `A Tour of Go` 的 `basics` 顺序逐步练习 Go 基础语法
- 保存每个题目的手写示例、实验代码和自己的理解
- 为后续 `flowcontrol`、`moretypes`、`methods`、`concurrency` 打基础

每个文件顶部都预留了：

- 中文题目说明
- 官方页面链接
- 题目列表来源
- 简单练习提示

## 使用建议

- 每次只打开一个题目文件，自己先写，再对照官方内容
- 写完后补一两行注释，记录你自己的理解
- 不要只改官方示例，尽量自己再加一个小变化
- 每完成几个题目，就关掉资料重写一次
