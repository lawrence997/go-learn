# flowcontrol

这个目录用于存放 [A Tour of Go - List](https://go.dev/tour/list) 中 `flowcontrol` 部分的跟练代码。

## 目录用途

- 练习 Go 中最常见的流程控制语法
- 熟悉 `for`、`if`、`switch`、`defer` 的实际写法
- 为后续函数封装、错误处理和工程代码阅读打基础

## 当前已预留的练习文件

- `for.go`
- `for-continued.go`
- `for-is-gos-while.go`
- `forever.go`
- `if.go`
- `if-with-a-short-statement.go`
- `if-and-else.go`
- `exercise-loops-and-functions.go`
- `switch.go`
- `switch-evaluation-order.go`
- `switch-with-no-condition.go`
- `defer.go`
- `stacking-defers.go`

每个文件顶部都预留了：

- 中文题目说明
- 官方页面链接
- 题目列表来源
- 简单练习提示

## 使用建议

- 先自己写出最小可运行例子，再对照官方内容
- 除了跑通示例，也要自己改一个条件或循环逻辑
- `exercise` 类文件尽量独立完成，不要直接照抄答案
